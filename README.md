# Legend of the Fallen Warrior

Idle/incremental browser RPG. Vanilla JavaScript (ES modules) bundled with Vite;
Bootstrap 3 + jQuery for styling and widgets.

- **Code map / where to find what:** [ARCHITECTURE.md](ARCHITECTURE.md)
- **Contributor / AI-assistant conventions:** [CLAUDE.md](CLAUDE.md)

## Quick start

```
npm install
npm run dev      # http://localhost:5173
```

`npm run build` → production bundle in `dist/` · `npm test` → Vitest suite ·
`npm run lint` / `npm run format` → ESLint / Prettier.
